## Privacy

Before publishing, all exported customer data is automatically sanitized.

Sensitive information removed:

- Buyer Username
- Receiver Name
- Phone Number
- Shipping Address
- Postal Code

Customers are replaced with pseudonyms such as

```
Customer-eb3fdf03
```

No personal information is exposed in the public dashboard.
